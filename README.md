# Phaser3 Memory Game

This is a minimal memory game implementation using [Phaser3](https://phaser.io)
Running example: [https://thawing-wildwood-59997.herokuapp.com/](https://thawing-wildwood-59997.herokuapp.com/)

## How to run

```bash
npm install
npm run start:dev
```
